## Introducción a SCM

La Gestión de la Configuración de Software (SCM) es una disciplina transversal en la ingeniería de software que administra la evolución de productos complejos. Debido a que el software es un activo intangible y en constante cambio, la SCM actúa como el marco que organiza y controla cada modificación. Su importancia fundamental reside en que permite la colaboración de múltiples integrantes sin comprometer la integridad del sistema, asegurando que siempre exista una línea base (baseline) estable y recuperable.
